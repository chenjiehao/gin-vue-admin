## Context

Phase 13 Wave 1 完成后，后端将提供 GET /dataSource/getTables 接口。

## Approach

**技术方案：**
- 复用现有 el-select 组件，新增"源表"/"目标表"下拉框
- 选择数据源后自动调用 API 加载表列表
- 切换数据源时清空已选的表

