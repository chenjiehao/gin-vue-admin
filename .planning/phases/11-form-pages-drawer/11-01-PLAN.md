## Goal

将"新增/编辑"操作从全页路由跳转改为在列表页右侧 div 区域局部加载，左侧列表保持显示，右侧动态切换列表/表单。

## Design

**页面布局：**
```
+-------------------+------------------------+
|   搜索区域         |                        |
+-------------------+------------------------+
|                   |                        |
|   数据表格         |    表单区域            |
|   (左侧)           |    (右侧,初始隐藏)     |
|                   |                        |
+-------------------+------------------------+
|   分页            |                        |
+-------------------+------------------------+
```

**交互逻辑：**
- 初始状态：右侧区域隐藏，左侧占满全宽
- 点击"新增"：右侧显示表单，左侧宽度收缩
- 点击"编辑"：右侧显示表单（预填数据），左侧收缩
- 表单提交/取消/返回：右侧隐藏，左侧恢复全宽

## Tasks

### Task 1: 重构 offlineSync.vue — 列表页内嵌表单

<read_first>
- web/src/view/dataIntegration/offlineSync.vue — 当前列表页完整代码
- web/src/view/dataIntegration/offlineSyncForm.vue — 表单内容（需要内嵌的部分）
</read_first>

<acceptance_criteria>
- 页面初始加载只显示搜索+表格+分页，右侧区域隐藏
- 点击"新增任务"按钮，右侧区域显示表单，左侧表格区域收缩
- 点击"编辑"按钮，右侧显示预填数据的表单
- 表单底部有"取消"按钮，点击返回列表视图
- 表单提交成功后自动返回列表视图
</acceptance_criteria>

<action>
1. 在 `offlineSync.vue` 的 template 中，将整个内容包裹在一个 flex 容器中：
   - 左侧 div（class="list-panel"）：包含搜索区域、表格、分页
   - 右侧 div（class="form-panel"）：初始 v-show="formVisible"，包含从 offlineSyncForm.vue 提取的表单内容

2. 添加响应式状态：
   ```javascript
   const formVisible = ref(false)
   const currentFormData = ref(null)

   const goCreate = () => {
     currentFormData.value = null
     formVisible.value = true
   }

   const goEdit = (row) => {
     currentFormData.value = { ...row }
     formVisible.value = true
   }

   const closeForm = () => {
     formVisible.value = false
     currentFormData.value = null
   }
   ```

3. 表单提交成功后调用 `closeForm()` 返回列表

4. 添加样式：
   ```css
   .page-container {
     display: flex;
     gap: 16px;
   }
   .list-panel {
     flex: 1;
     transition: all 0.3s;
   }
   .list-panel.collapsed {
     flex: 0 0 60%;
   }
   .form-panel {
     flex: 1;
     transition: all 0.3s;
     display: none;
   }
   .form-panel.visible {
     display: block;
   }
   ```

5. 将 offlineSyncForm.vue 的表单 HTML 和逻辑内联到 offlineSync.vue 的 form-panel 中，保留所有表单字段和验证逻辑
</action>
